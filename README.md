# NatureMiniQuests

A mobile app (iOS + Android) that encourages outdoor exploration in Toronto through admin-curated mini quests.

This repository contains the **product plan and documentation**. The technology stack has not yet been decided — the docs are intentionally implementation-agnostic so that different frameworks, platforms, and backend options can be evaluated before any commitment is made.

See the [docs/](./docs/README.md) folder for the full project documentation.
