# Don Valley / Evergreen Brick Works — Quest Content (MVP)

This is the **first content pack** for Toronto Nature Mini Quests.

## Design goals for this pack
- GPS-friendly (open sky when possible)
- Safe, public-access checkpoints
- Short loop (30–60 minutes) for MVP validation
- Checkpoints are **in order**, manual check-in
